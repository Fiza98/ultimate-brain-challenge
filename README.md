# Ultimate Brain Challenge  

Welcome to **Ultimate Brain Challenge**, an interactive and fun quiz game designed to test your knowledge in science, math, and logic. This project combines education with engaging animations, sound effects, and a leaderboard to make learning exciting for all ages!  

## 🎯 Overview  

**Ultimate Brain Challenge** is a web-based quiz game that presents a series of thought-provoking questions. Players select answers, receive immediate feedback with sound effects, and track their performance through a leaderboard system.  

## ✨ Features  

- **🧠 Brain-Boosting Questions** – Challenge yourself with science, math, and logic-based quizzes.  
- **🎵 Engaging Experience** – Enjoy sound effects, background music, and smooth animations.  
- **📊 Leaderboard** – Keep track of high scores and compete with others.  
- **💡 Adaptive Learning** – Some questions are open-ended to understand learning preferences.  
- **🌐 Fully Responsive** – Works seamlessly on both desktop and mobile devices.  

## 🚀 Technology Stack  

This project is built using:  
- **React** (Interactive UI)  
- **Tailwind CSS** (Stylish and responsive design)  
- **Framer Motion** (Smooth animations)  
- **JavaScript** (Game logic and interactivity)  
- **Audio API** (Background music & sound effects)  

## 🎮 How to Play  

1. Visit the live version of **Ultimate Brain Challenge** [here](https://fiza98.github.io/ultimate-brain-challenge).  
2. Answer the questions by selecting the correct option.  
3. Get instant feedback with animations and sounds.  
4. See your score and compare with others on the leaderboard.  
5. Play again to improve your knowledge and score!  

## 🛠 How to Set Up Locally  

1. **Clone the Repository:**  
   ```sh
   git clone https://github.com/fiza98/ultimate-brain-challenge.git
   
2. **Navigate into the Project Directory:**
   ```sh
   cd ultimate-brain-challenge

3. **Install Dependencies**
   ```sh
   npm install

4. **Run the Project Locally:**
   ```sh
   npm start

5. **Open `http://localhost:3000/` in your browser to play the game locally.**
